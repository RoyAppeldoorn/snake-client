# Vuesnake!

A websocket based multiplayer snake game made with Vue.js

## What is Vuesnake?

Vuesnake is a Vue.js based multiplayer snake game, using websockets as the communication. Vuesnake is connected to a small gameserver [I'm an inline-style link with title](https://github.com/RoyAppeldoorn/snake-server "written in java")
